# UI 合約：投籃遊戲

**版本**：1.0.0  
**產出日期**：2026-03-13  
**適用規格**：`specs/001-basketball-game/spec.md`

---

## 概述

本文件定義投籃遊戲前端靜態網站的 UI 合約，包含畫面狀態、顯示元素與使用者互動規範。遊戲為純前端應用，無後端 API，因此以 UI 合約取代 REST/GraphQL 合約。

---

## 畫面狀態（Screen States）

### 1. 遊戲主畫面（PLAYING）

**觸發條件**：頁面載入後立即顯示（初始狀態）

**必要顯示元素**：

| 元素 | 說明 | 位置 |
|------|------|------|
| Canvas（遊戲畫布） | 渲染籃球、籃框、飛行動畫 | 頁面主體，全寬或固定比例 |
| 得分板（Score） | 格式：`得分：{score}` | 畫布左上或右上 |
| 剩餘球數（Shots Left） | 格式：`剩餘球數：{shotsLeft}` | 得分板旁 |
| 籃框（Hoop） | 固定位置，含左右框邊與後板 | 畫布右上 1/3 區域 |
| 籃球（Ball） | 待投狀態，顯示於起始位置 | 畫布底部中央 |
| 瞄準線（Aim Line） | 拖曳時動態顯示投球方向 | 跟隨滑鼠/觸控 |

---

### 2. 球飛行中（FLYING）

**觸發條件**：玩家放開滑鼠/觸控後

**狀態規範**：
- 球沿拋物線移動，`GRAVITY` + `DAMPING` 控制
- 輸入操作被忽略（防止重複投球）
- 若球通過籃框偵測區域 → 觸發「進球事件」
- 若球超出畫布邊界（底部） → 觸發「未進事件」

---

### 3. 進球特效（SCORED）

**觸發條件**：`isBallInHoop()` 回傳 `true`

**必要顯示元素**：
- 視覺特效（閃光或動畫，持續約 0.5 秒）
- 得分板即時更新 `+1`
- 球重置至起始位置

---

### 4. 遊戲結束畫面（OVER）

**觸發條件**：`shotsUsed === totalShots`（10 球全部投完）

**必要顯示元素**：

| 元素 | 說明 |
|------|------|
| 標題 | `「遊戲結束」` |
| 本局得分 | `得分：{score} / {totalShots}` |
| 命中率 | `命中率：{hitRate}%` |
| 最高分 | `最高紀錄：{highScore}` |
| 破紀錄提示（條件顯示） | 本局得分 > 歷史最高分時顯示「🎉 新紀錄！」 |
| 再玩一次按鈕 | 文字：`「再玩一次」`，點擊後重置遊戲 |

## 使用者互動合約

### 投籃操作（滑鼠 / 觸控）

| 事件 | 說明 | 條件 |
|------|------|------|
| `pointerdown` | 開始瞄準，記錄起始座標 | `!ball.isFlying && !game.isOver` |
| `pointermove` | 更新瞄準線終點 | `inputState.isAiming === true` |
| `pointerup` | 計算投射速度，開始飛行 | `inputState.isAiming === true` |

**速度計算公式**：
```
vx = (currentX - startX) * VELOCITY_MULTIPLIER
vy = -(currentY - startY) * VELOCITY_MULTIPLIER
```
（反向 Y 軸：向下拖 → 向上投）

---

### 再玩一次按鈕

| 事件 | 說明 |
|------|------|
| `click` | 重置 `GameSession`（score=0, shotsUsed=0, hits=0, isOver=false），球回初始位置，遊戲回到 PLAYING 狀態 |

## localStorage 合約

| 鍵值 | 型別 | 說明 |
|------|------|------|
| `basketball_high_score` | `string`（JSON 序列化整數） | 歷史最高分，整數，最小值 0 |

**讀取失敗（localStorage 不可用）**：靜默退化，使用記憶體暫存值 `0`，不顯示錯誤。

## 效能合約（對應成功指標）

| 指標 | 目標值 | 驗證方式 |
|------|--------|----------|
| 首次互動時間 | 頁面載入後 < 3 秒可投球 | Playwright 腳本計時 |
| 動畫幀率 | ≥ 30 fps | Chrome DevTools Performance panel |
| 進球判斷準確率 | 100%（進必得分，未進絕不誤判） | unit 測試窮舉邊界 |
